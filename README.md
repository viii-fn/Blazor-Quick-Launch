BlazorQL is a quick launch alternative to run blazor web assmebly applications without the installing the C# DevKit.

Place the "vercel.json" file in yout wwwroot folder.

Create a ".vscode" folder and place the "tasks.json" and "launch.json" files inside it

Replace the "project-name.sln" solution file name with you csproj's file name. Also edit the "project-name.sln" file (Replace all "project-name" in the solution file with your csproj's file name)
